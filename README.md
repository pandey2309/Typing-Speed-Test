# TypeFlow - Typing Speed Test

A simple and interactive web app to measure typing speed (WPM) and accuracy in real-time.

## Features

- Real-time typing speed calculation (WPM)
- Accuracy tracking
- Restart test anytime
- Light/Dark mode toggle
- Dynamic UI feedback while typing

## Tech Stack

- Python (Flask)
- HTML
- CSS
- JavaScript

## Project Structure
```
 Typing-Speed-Test/
│
├── app.py
├── templates/
│   └── index.html
├── static/
│   ├── style.css
│   ├── script.js
│   └── words.js
```


## How to Run
1. Install Flask:
     pip install flask
   
2. Run the app:
     python app.py
   
3. Open in browser:
     http://127.0.0.1:5000/
