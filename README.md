# Typing-Speed-Test

A simple and interactive web app to measure typing speed (WPM) and accuracy in real-time.

## Features

- Real-time typing speed calculation (WPM)
- Accuracy tracking
- Restart test anytime
- Light/Dark mode toggle
- Dynamic UI feedback while typing

## Tech Stack

- Python (Flask)
- HTML
- CSS
- JavaScript
